# os24fall-stu

## Inrto

上游仓库: 浙江大学操作系统课程学生版仓库[os24fall-stu](https://github.com/ZJU-SEC/os24fall-stu)

本仓库为个人实现，包括本学期 lab1 - lab6 完成的所有代码。其中 src/labn 为实验 n 的指导代码, src/labn_ 为完整的实验实现代码, 里面包含了编译完成的内核文件。

本仓库于 24.12.31 public，仅供参考。
